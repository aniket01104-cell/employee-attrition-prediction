# 📊 Employee Attrition Prediction Project

## 🚀 Project Overview

This is an end-to-end project that predicts employee attrition using machine learning, analyzes workforce trends using SQL, visualizes HR insights through Power BI dashboards, and provides real-time attrition predictions using an interactive Flask web application.

The goal is to help HR teams identify employees who are likely to leave and take proactive retention actions.

\---

## 🎯 Objectives

* Predict employee attrition using ML models
* Analyze attrition trends using SQL
* Build interactive dashboards using Power BI
* Deploy model using Flask
* Provide actionable retention recommendations

\---

## 🧠 Machine Learning Model

* Model: XGBoost Classifier
* Output: Attrition Probability (Low / Medium / High Risk)
* Explainability: SHAP
* Features: Age, Monthly Income, Overtime, Job Role, Department, Distance From Home, Work Life Balance, Job Satisfaction, Years at Company, etc.

\---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Scikit-learn)
* XGBoost
* SHAP
* MySQL (Data Analysis)
* Power BI (Dashboard)
* Flask (Deployment)
* HTML/CSS

\---

## 📊 Key Insights

* Employees working overtime show higher attrition risk
* Lower salary employees are more likely to leave
* Poor work-life balance increases attrition
* Employees in early tenure stages leave more frequently
* Certain departments show higher attrition rates
